Quick-start Instructions

# Windows users: Click Download, unzip and run jar

~ $ wget http://tinyurl.com/h2gis-zip -O h2gis.zip

~ $ unzip h2gis.zip

~ $ cd h2gis-standalone

~/h2gis-standalone $ ./run.sh

# => Click "Connect" in the web interface
